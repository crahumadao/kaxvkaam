Para entender esta herramienta

Este es un texto de ejemplo para que entiendas como usar esta herramienta
Azümam tÜfachi küzawpeyüm

Tüfa kiñe wirintukun tami azümafiel chumgechi püneafiel tüfachi küzawpeyüm

wazkü = vi hervir; i = El/Ella

Esta palabra se puede traducir como (el/ella/es@) "hirvió"

amu = vi ire, andar, continuar, avanzar ; a = futuro ; iñ = Nosotros (mas de 2 personas)

Esta palabra se puede traducir como "Nosotros iremos"

ayüwküle = vi estar alegre, estar contento ; n = Yo /o el verbo "puro"

Esta palabra se puede traducir como "Estoy contento" o "Estar contento"

Hola hermanos hermanas. Yo me llamo Juan. Conversaremos de mi descanso de invierno (vacaciones). Yo fui al sur, por Gorbea. Fui a estudiar en un koneltun*(internado de mapuzugun). Yo voy a temuco habitualmente, pero este año no fui, en el verano iré a ver a mi familia a temico. Cuando estaba en Gorbea, hable con algunas personas, ell@s también estudian mapuzugun. Otras personas saben hablar mapuzugun. Ahora yo se más mapuzugun, así que todavía estoy feliz
